# Warkopsaja Frontend

Hotspot payment website for Pulau Balang Lompo.

## How it works
- Displays WiFi packages from MikroTik (via backend API)
- Processes payments through Midtrans
- Creates hotspot accounts after successful payment

## Deployment
This is deployed via Coolify using Docker Compose.

## Edit the website
Edit `public/index.html` to change the website design, text, or features.
Push to the `production` branch and Coolify will auto-redeploy.
